# 工作流自动化工具选型报告
## 调研时间: 2026-03-05

## 一、主流自动化工具对比

### 1. Zapier
- **特点**: 最成熟的SaaS自动化平台
- **优点**: 
  - 集成5000+应用
  - 无需编程
  - 界面友好
- **缺点**:
  - 免费版限制严格
  - 国内应用集成较少
  - 数据需经过美国服务器

### 2. n8n (开源)
- **特点**: 开源自动化工作流引擎
- **优点**:
  - 完全自托管，数据安全
  - 社区活跃，节点丰富
  - 可Docker部署
- **缺点**:
  - 需要技术能力部署维护
  - 付费版功能才完整

### 3. Make (原Integromat)
- **特点**: 可视化流程编排能力强
- **优点**:
  - 界面直观
  - 复杂逻辑支持好
  - 性价比高
- **缺点**:
  - 学习曲线较陡
  - 国内生态一般

### 4. 飞书自动化
- **特点**: 集成飞书生态
- **优点**:
  - 与飞书深度集成
  - 国产方案，数据合规
  - 支持多维表格/审批等
- **缺点**:
  - 生态相对封闭
  - 扩展性有限

## 二、选型建议

### 场景1: 个人/小团队轻量自动化
- **推荐**: 飞书自动化 / Zapier
- **理由**: 零代码，部署快

### 场景2: 企业级自动化
- **推荐**: n8n自托管
- **理由**: 数据可控，可定制

### 场景3: 国产化需求
- **推荐**: 飞书自动化 + 自建服务
- **理由**: 满足数据合规要求

## 三、当前可优化方向

1. **飞书多维表格自动化**: 任务状态变更自动通知
2. **定时任务**: 每日自动生成/更新任务
3. **数据同步**: 跨平台数据自动汇总
