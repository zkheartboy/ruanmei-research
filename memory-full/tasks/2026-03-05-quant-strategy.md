# A股均线策略研究报告
## 研究时间: 2026-03-05

## 一、常用均线指标

### 1. 简单移动平均 (SMA/MA)
- 计算方式: N日收盘价之和/N
- 常用周期: 5, 10, 20, 60, 120, 250日
- 特点: 信号滞后，但稳定

### 2. 指数移动平均 (EMA)
- 计算方式: 加权平均，近期权重更高
- 常用周期: 12, 26日（MACD基础）
- 特点: 对价格变化更敏感

### 3. 布林带 (Bollinger Bands)
- 组成: 中轨(20日EMA) ± 2倍标准差
- 策略: 突破上轨卖出，突破下轨买入
- 特点: 适合震荡市场

### 4. 均线交叉策略
- **金叉**: 短期均线上穿长期均线 → 买入信号
- **死叉**: 短期均线下穿长期均线 → 卖出信号
- 常用组合: 5/20, 10/60, 20/120

## 二、策略有效性分析

### 牛市环境
- 均线策略表现较好，主要得益于趋势跟随
- 推荐: 60日/120日均线组合

### 熊市环境
- 均线策略频繁失效，产生大量假信号
- 建议: 结合波动率指标过滤

### 震荡市场
- 布林带策略相对有效
- 建议: 缩小周期，增加交易频率

### 优化方向
1. **自适应周期**: 根据市场状态动态调整均线周期
2. **多指标过滤**: 结合RSI、MACD过滤假信号
3. **仓位管理**: 根据趋势强度调整仓位

## 四、风险提示

- 均线策略是趋势跟踪策略，在震荡市中表现不佳
- 需要配合止损和仓位管理
- 历史回测不代表未来收益
