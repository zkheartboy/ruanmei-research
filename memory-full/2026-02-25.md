### GitHub检索方法研究

从停云（tingyun）的代码中学习到的GitHub检索方法：

**使用的API：**
1. https://ghapi.huchen.dev/repositories?since=daily
2. https://api.gitterapp.com/repositories?since=daily

**参数：**
- since=daily (今日趋势)
- since=weekly (本周趋势)
- since=monthly (本月趋势)

**返回字段：**
- author - 作者/组织
- name - 项目名
- description - 描述
- language - 编程语言
- stars - 总star数
- currentPeriodStars - 本期增长
- url - GitHub链接

**容错机制：**
- 第一个API失败自动尝试第二个
- 两个都失败则使用内置演示数据
