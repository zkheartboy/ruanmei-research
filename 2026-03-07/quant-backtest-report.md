## 一、智能定投策略原理

### 1.1 传统定投 vs 智能定投

| 维度 | 传统定投 | 智能定投 |
|------|----------|----------|
| 金额 | 固定金额 | 动态调节 |
| 时机 | 无脑扣款 | 估值择时 |
| 逻辑 | 平均成本 | 低买高卖 |

### 1.2 核心策略指标

**估值百分位 (PE/PB分位)**
- PE分位 < 30%：加大定投（低估）
- PE分位 30-70%：正常定投
- PE分位 > 70%：减少定投（高估）

**均线偏离度**
- 指数跌破60日均线：加大定投
- 指数站上60日均线：正常定投

## 三、策略实现框架

### 3.1 Python回测代码结构

```python
# 智能定投策略框架
class SmartDCA:
    def __init__(self):
        self.pe_percentile = 0  # PE百分位
        self.ma_deviation = 0  # 均线偏离度
    
    def calculate_position(self):
        """计算定投仓位"""
        if self.pe_percentile < 30:
            return 2.0  # 双倍定投
        elif self.pe_percentile < 70:
            return 1.0  # 正常定投
        else:
            return 0.5  # 减半定投
    
    def should_rebalance(self, portfolio_value, target_value):
        """判断是否需要再平衡"""
        return abs(portfolio_value - target_value) / target_value > 0.1
```
