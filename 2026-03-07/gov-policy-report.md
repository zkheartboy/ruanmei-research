## 一、政策背景与演进

### 1.1 政策历程

| 时间 | 事件 |
|------|------|
| 2024年1月 | 国务院印发《关于加快推进"高效办成一件事"的指导意见》 |
| 2024年 | 推出首批重点事项清单 |
| 2025年1月 | 国务院部署2025年度第一批12个重点事项 |
| 2025年7月 | 国务院部署2025年度第二批9个重点事项 |
| 2025年12月 | 累计推出四批重点事项清单 |
| 2026年2月 | 建立常态化推进机制 |

### 1.2 2025年度重点事项

**第一批（12项）**
- 新生儿出生、入学、退休、身后等"一件事"
- 企业开办、变更、注销"一件事"
- 信用修复、企业融资等

**第二批（9项）**
- 高效办成电解铝产能置换一件事
- 危险化学品经营许可一件事
- 排污权交易一件事

## 二、核心改革举措

### 2.1 集成化办理

**改革要点：**
- 将多个相关联的"单项事"整合成"一件事"
- 实现"一次告知、一表申请、一套材料、一窗（端）受理、一网办理"

**实现路径：**
```
流程再造 → 事项整合 → 数据共享 → 协同办理
```

### 2.2 减证便民

**主要措施：**
- 梳理证明材料清单
- 推行告知承诺制
- 电子证照互通互认

**典型案例：**
- "无证明城市"建设
- 电子证照"免证办"

### 2.3 "一网通办"

**平台建设：**
- 全国一体化政务服务平台
- 地方政务服务APP/小程序
- 政务服务"跨省通办"

### 2.4 AI赋能政务

**2025年新技术应用：**
- 智能问答（政策解读）
- 智能引导（办事流程）
- 智能预填（表单填写）
- 智能帮办（全程代办）
- 智能审批（辅助审核）
